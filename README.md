# Lokaverkefni Netverslun

A React, TypeScript and Supabase webshop project for a final assignment.

## Features

- Home page
- Product list
- Product detail page
- Product search
- Category filtering
- Supabase product database
- Supabase authentication
- Cart with Zustand state management
- Add, remove and update product quantity in cart
- Protected checkout page
- Fake checkout / fake payment flow
- Confirmation page after fake payment
- Responsive layout
- Tests for cart logic and fake checkout logic

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- Supabase
- React Query
- Zustand
- Vitest
- CSS
